# json-diff-tool

轻量的 JSON 格式化与多组 Diff 工作区。

它解决的是一个很具体的问题：很多时候你不是只想“看一段 JSON”，而是想先把 JSON 整理清楚，再连续对比多组请求、响应或配置差异。这个工具把这两件事放在一个页面里，尽量减少切换和视觉干扰。

## 能做什么

- 格式化单段 JSON，并以可折叠树结构查看
- 在工作区中管理多组 JSON 对比
- 自动刷新差异结果
- 只看差异，减少无关节点干扰
- 把常用对比组保存到本地，刷新页面后仍可继续

## 适合谁

- 需要频繁检查接口返回的开发者
- 需要对比配置、请求参数、响应内容的人
- 想用一个简单页面长期保存几组 JSON 对比的人

## 快速开始

```bash
npm install
npm run dev
```

本地开发服务启动后，直接在浏览器打开 Vite 输出的地址即可。

## 使用方式

1. 在“格式化 / Viewer”里粘贴 JSON，先完成格式化和结构检查。
2. 进入“对比工作区”，新建一个组。
3. 分别填入左侧和右侧 JSON，Diff 会自动刷新。
4. 需要长期保留时，把该组保存到本地。

## 构建

```bash
npm run build
```

构建结果输出到 `dist/`，可直接部署到静态托管平台，例如 Vercel。

## 设计取向

- 先格式化，再对比
- 快速输入，快速观察
- 默认保持页面干净，把注意力留给差异本身
